Public projects and things that I've made and remembered to add here. You can find a full list on my [GitHub](https://github.com/Rjbaird "Ryan Baird's Github").

## websites

**[Ryanbaird.com](https://ryanbaird.com/)**

My personal website and portfolio. You'll find links to my (rarely used) social media accounts, contact information and links articles and projects. Built with [Astro](https://astro.build/ "Astro"), [TailwindCSS](https://tailwindcss.com/ "Tailwind CSS") & [MDX](https://mdxjs.com/ "MDX").

**[Transaction Fee Calculator](https://transaction-fee-calculator.netlify.app/)**

A calcualtor to help you find out how much you'll make after card processing fees. You can add products, adjust the rates from your provider and even export everything as JSON. All data is stored locally. Built with [SolidJS](https://www.solidjs.com/ "Solid JS") and [TailwindCSS](https://tailwindcss.com/ "Tailwind CSS").

**[Weather Dashboard](https://github.com/Rjbaird/WeatherDashboard)**

Search current and future weather conditions using the OpenWeather API. You'll recieve the following information: Temperature, Humidity, Wind Speed and UV Index. Oh and a gif from the Giphy API!

## apis

**[Shonen Jump API](https://github.com/Rjbaird/shonen-jump-api)**

An unofficial REST API showing data about English translations from [Weekly Shonen Jump](https://www.viz.com/shonenjump "Weekly Shonen Jump"). Built with Typescript, Express, MongoDB. Currently being moved off Heroku.
