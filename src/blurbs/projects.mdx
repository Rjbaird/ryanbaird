# apis
**[Shonen Jump API](https://github.com/Rjbaird/shonen-jump-api)**

An unofficial REST API showing data about English translations from [Weekly Shonen Jump](https://www.viz.com/shonenjump). Built with Typescript, Express, MongoDB. Currently being moved off Heroku.

# other
**[Coinbase & YNAB Sync](https://github.com/Rjbaird/YNAB-Coinbase-Sync)**

A python script that uses the [You Need A Budget](https://www.youneedabudget.com/) and [Coinbase](https://www.coinbase.com/) APIs to automatically update an investment or tracking account in YNAB

**[Weather Dashboard](https://github.com/Rjbaird/WeatherDashboard)**

Search current and future weather conditions using the OpenWeather API. You'll recieve the following information: Temperature, Humidity, Wind Speed and UV Index. Oh and a gif from the Giphy API!
